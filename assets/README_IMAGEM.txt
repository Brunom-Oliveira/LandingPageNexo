GUIA DE IMAGEM PARA REDES SOCIAIS (WHATSAPP/FACEBOOK)

1. Para que seu site apareça bonito quando compartilhado, você precisa de uma imagem.
2. Tamanho Recomendado: 1200 pixels de largura por 630 pixels de altura (Formato JPG ou PNG).
3. Onde conseguir uma imagem gratuita e bonita:
   - Unsplash.com (Busque por "Office", "Startup", "Code")
   - Pexels.com
   - Canva (Crie um design com esse tamanho)

4. Como Salvar:
   - Renomeie a imagem para: share-image.jpg
   - Coloque dentro desta pasta: c:\Users\suporte\landingpag\assets\

5. Depois de salvar:
   - Rode os comandos do Git para enviar para o servidor:
     git add assets/share-image.jpg
     git commit -m "add social share image"
     git push origin main

   - E atualize na VPS.

Pronto! Seu site vai ter uma "capa" profissional nas redes sociais.
